# Native Elements

## Overview

Native-element minerals are crystals of a single element. In
gemology, the most important native element is diamond (carbon),
formed by sp³-hybridized carbon atoms in a tetrahedral covalent
network — the hardest known natural material. Other native
elements (gold, silver, platinum) are typically used as
ornamental or precious-metal work rather than faceted gems.


## Formula Class

Single-element composition

## Crystal Structure

Atoms occupy lattice positions in characteristic geometries;
metallic native elements (gold, silver) form face-centered
cubic structures, while non-metals (diamond) form covalent
tetrahedral networks.
