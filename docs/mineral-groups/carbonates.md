# Carbonates

## Overview

Carbonates include several distinctive gem families:
- **Calcite subgroup**: calcite (incl. Iceland spar),
  rhodochrosite, magnesite, siderite
- **Aragonite subgroup**: aragonite, pearl mother-of-pearl,
  coral
- **Dolomite, malachite, azurite**: complex hydrated or
  copper-bearing carbonates
Most carbonates are soft (Mohs 3–4) and used as carving or
ornamental material.


## Formula Class

CO₃²⁻ + divalent metal cations

## Crystal Structure

Planar [CO₃]²⁻ triangular groups coordinate with divalent metal
cations (Ca, Mg, Mn, Fe, Cu, Zn, Ba, Sr). Calcite subgroup:
trigonal; aragonite subgroup: orthorhombic.
