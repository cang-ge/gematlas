# Triclinic

## Overview

The triclinic system has the lowest symmetry: no axis is
perpendicular to any other, and no angles are 90°. Crystals are
biaxial. Kyanite's distinctive hardness anisotropy (4–5 across vs.
6–7 along) is a textbook example of triclinic structural anisotropy.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a ≠ b ≠ c |
| Interaxial angles | α ≠ β ≠ γ ≠ 90° |
| Symmetry tier | lowest |

| Optic sign | + or − |

| Cleavage | variable / poor |
