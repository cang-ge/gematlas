# Hexagonal

## Overview

The hexagonal system (in strict 4-index notation) has three equal
horizontal axes 120° apart and one perpendicular c-axis. Crystals
are uniaxial. Beryl, with its iconic six-sided prismatic habit,
is the flagship species. Note: many references group trigonal
with hexagonal as a single 6-fold system.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a₁ = a₂ = a₃ ≠ c (4-index notation) |
| Interaxial angles | 120° / 120° / 120° between lateral axes, 90° vertical |
| Symmetry tier | medium-high |
| Optical class | uniaxial |
| Optic sign | + or − |
| Birefringence | 0.000–0.180 |
| Cleavage | poor / none |
| Habit | Hexagonal prisms, bipyramids |
