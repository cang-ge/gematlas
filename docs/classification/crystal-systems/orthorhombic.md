# Orthorhombic

## Overview

The orthorhombic system has three mutually perpendicular but
unequal axes. Crystals are biaxial with two optic axes; the optic
sign (positive or negative) varies by species. Many gem species,
including the chrysoberyl and olivine groups, belong here.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a ≠ b ≠ c |
| Interaxial angles | α = β = γ = 90° |
| Symmetry tier | medium |
| Optical class | biaxial |
| Optic sign | + or − |
| Birefringence | 0.040–0.180 |
| Cleavage | good parallel to one or two pinacoids |
| Habit | Rhombic prisms, dipyramids, tabular |
