# Cubic (Isometric)

## Overview

The cubic (also called isometric) system has the highest symmetry
of the seven crystal systems. Three axes of equal length intersect
at right angles. Cubic crystals are optically isotropic — light
travels at the same velocity in every direction, producing no
birefringence and no pleochroism. Identification therefore relies
on refractive index, specific gravity, and spectroscopic tests.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a = b = c |
| Interaxial angles | α = β = γ = 90° |
| Symmetry tier | highest |
| Optical class | isotropic |
| Optic sign | none |
| Birefringence | none |
| Cleavage | variable by species |
| Habit | Cubes, octahedra, dodecahedra, granular masses |
