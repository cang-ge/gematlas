# Monoclinic

## Overview

The monoclinic system has three unequal axes with two meeting at
right angles and the third (b-axis) inclined. Crystals are
biaxial. The dominant cleavage is parallel to the (010)
pinacoid. Jade (nephrite) and most feldspars fall here.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a ≠ b ≠ c |
| Interaxial angles | α = γ = 90°, β ≠ 90° |
| Symmetry tier | low-medium |
| Optical class | biaxial |
| Optic sign | + or − |
| Birefringence | 0.015–0.105 |
| Cleavage | pinacoidal (single good cleavage) |
| Habit | Oblique prisms, tabular, often twinned |
