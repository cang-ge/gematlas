# Trigonal

## Overview

The trigonal system has a 3-fold rather than 6-fold principal
axis (no horizontal mirror plane perpendicular to it). Crystals
are uniaxial. Corundum, quartz, and tourmaline are the most
important gem families. Twinning is very common and useful for
identification (e.g., Dauphiné / Japan law in quartz).


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a₁ = a₂ = a₃ ≠ c (rhombohedral subtype) |
| Interaxial angles | as hexagonal |
| Symmetry tier | medium-high |
| Optical class | uniaxial |
| Optic sign | + or − (varies) |
| Birefringence | 0.000–0.116 |
| Cleavage | variable |
| Habit | Rhombohedra, scalenohedra, prismatic with striations |
