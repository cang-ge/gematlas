# Tetragonal

## Overview

The tetragonal system has two equal horizontal axes and one
longer or shorter vertical axis. Crystals are optically uniaxial
with a single optic axis parallel to the c-axis; the optic sign
is positive in most gem species (negative in a few). The c-axis
is the "extraordinary" refractive index direction.


## Crystallographic Parameters

| Property | Value |
|---|---|
| Axial lengths | a = b ≠ c |
| Interaxial angles | α = β = γ = 90° |
| Symmetry tier | medium-high |
| Optical class | uniaxial |
| Optic sign | + |
| Birefringence | 0.040–0.300 |
| Cleavage | parallel to base (001) |
| Habit | Square prisms, four-sided bipyramids, tabular |
