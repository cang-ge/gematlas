## Classification

| Property | Value |
|---|---|
| Mineral Family | Zircon |
| Formula | ZrSiO₄ |
| Crystal System | tetragonal |

## Physical Properties

| Property | Value |
|---|---|
| Mohs Hardness | 7.25 |
| Specific Gravity | 4.65 |
| Refractive Index | 1.810-2.024 |

## Optical Properties

| Property | Value |
|---|---|
| Pleochroism | weak |
| Typical Colors | Blue (heated), Golden yellow, Colorless, Honey, Red |
| Color Cause | U/Th radiation damage + transition metal coloration; natural blue is rare and typically requires heating |

## Treatments & Disclosure

| Treatment | Details |
|-----------|---------|
| Common Methods | heat-treatment |
| Disclosure Required | Yes |
| Note | Commercial blue zircon is usually heat-treated brown-yellow rough; not to be confused with cubic zirconia (CZ) |
