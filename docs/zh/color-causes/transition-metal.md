# 过渡金属离子致色

## 概述

过渡金属离子（如 Cr、Fe、Mn、Co、Ni、V、Cu、Ti）含有未填满的 d
或 f 电子层。当这些离子进入晶格取代主晶格中的位置（如 Al、Si、
Mg）时，其能级被晶场劈裂为多组 d-d 跃迁。吸收可见光特定波段后，
透射或反射光中残留的颜色即为我们看到的颜色。这是宝石界最重要
的致色机制，约 70% 的有色宝石属此类。


## 致色机制

**晶场理论**：过渡金属离子周围配位阴离子（O²⁻、OH⁻、F⁻）产生
静电场，使简并的 d 轨道分裂为不同能级。电子在这些分裂能级之间
跃迁吸收特定波长光子。配位几何（八面体 vs 四面体）、键长、键角
决定能级差，从而决定吸收波长与最终颜色。


## 示例宝石

| 宝石 | 致色元素 | 化学式宿主 | 颜色 |
|---|---|---|---|
| Ruby / 红宝石 | Cr³⁺ | Al₂O₃ | red |
| Emerald / 祖母绿 | Cr³⁺ / V³⁺ | Be₃Al₂Si₆O₁₈ | green |
| Sapphire (blue) / 蓝宝石 | Fe²⁺–Ti⁴⁺ | Al₂O₃ | blue |
| Aquamarine / 海蓝宝 | Fe²⁺ | Be₃Al₂Si₆O₁₈ | blue |
| Peridot / 橄榄石 | Fe²⁺ | (Mg,Fe)₂SiO₄ | olive-green |
| Tsavorite / 沙弗莱 | V³⁺ / Cr³⁺ | Ca₃Al₂Si₃O₁₂ | green |
| Alexandrite / 亚历山大石 | Cr³⁺ | BeAl₂O₄ | green / red (color-change) |
| Rhodonite / 蔷薇辉石 | Mn²⁺ | MnSiO₃ | pink |
| Rhodochrosite / 菱锰矿 | Mn²⁺ | MnCO₃ | rose-pink |
