# 电荷转移

## 概述

电荷转移是电子在相邻离子或同一离子的不同能级之间跳跃吸收光子
的过程。与 d-d 跃迁相比，电荷转移的吸收强度通常高 10²–10⁴ 倍，
因此即使致色离子浓度很低也能产生饱和浓艳的颜色。常见类型：
**金属-金属电荷转移（MMCT）**、**氧-金属电荷转移（OMCT）**、
**金属-配体电荷转移（MLCT）**。


## 致色机制

**金属-金属电荷转移 (MMCT)**：相邻两个金属离子之间电子转移。
经典案例：蓝宝石的 Fe²⁺–Ti⁴⁺，一个电子从 Fe 跃迁到 Ti，吸收
黄红光 → 透射蓝光。
**氧-金属电荷转移 (OMCT)**：O²⁻ → 金属离子（如 Fe³⁺、Mn⁴⁺）
电子转移。常见于蓝宝石（Fe²⁺–O–Ti⁴⁺）和黄绿色蓝宝石。
**金属-配体电荷转移 (MLCT)**：常见于有机宝石（如珍珠质）。


## 示例宝石

| 宝石 | 转移对 | 化学式宿主 | 颜色 |
|---|---|---|---|
| Blue Sapphire / 蓝宝石 | Fe²⁺–Ti⁴⁺ | Al₂O₃ | blue |
| Amethyst / 紫晶 | Fe²⁺ → Fe⁴⁺ (via O²⁻) | SiO₂ | violet |
| Yellow Sapphire / 黄宝石 | Fe³⁺–O²⁻ | Al₂O₃ | yellow |
| Aquamarine (deep) / 海蓝宝 | Fe²⁺–Fe³⁺ | Be₃Al₂Si₆O₁₈ | deep blue |
| Hematite / 赤铁矿 | Fe²⁺–Fe³⁺ | Fe₂O₃ | red-brown |
| Kunzite / 孔赛石 | Mn²⁺ → Mn⁴⁺ | LiAlSi₂O₆ | pink (fades with light) |
