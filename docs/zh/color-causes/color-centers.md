# 色心致色

## 概述

色心是晶格中能选择性吸收可见光的点缺陷。当晶格缺少某个离子
（空位）、多出一个离子（间隙），或含有杂质离子改变局部电荷平衡
时，邻近的电子或空穴被束缚在缺陷处形成"色心"，其能级刚好落在
可见光波段。色心可通过天然辐射或人工辐照（如加热后辐照）产生，
也可能因日晒褪色，这是检测优化处理的关键判断点。


## 致色机制

常见色心类型：
- **F 心**（阴离子空位捕获电子）：常见于萤石、紫色石盐
- **V 心**（阳离子空位捕获空穴）：紫色石英（紫晶）
- **杂质色心**：杂质离子周围电子空位，如紫晶的 Fe⁴⁺ 色心
色心可在实验室通过辐照加速形成，但天然色心通常经过数百万年辐
射积累。辐照处理宝石的检测常依赖色心在短波紫外下的发光特征。


## 示例宝石

| 宝石 | 缺陷类型 | 化学式宿主 | 颜色 |
|---|---|---|---|
| Amethyst / 紫晶 | Fe⁴⁺ color center | SiO₂ | violet |
| Smoky Quartz / 烟晶 | Al impurity + e⁻ | SiO₂ | brown |
| Citrine / 黄水晶 | Fe³⁺ color center | SiO₂ | yellow |
| Fluorite / 萤石 | F-center (vacancy) | CaF₂ | purple / blue |
| Yellow Diamond / 黄钻 | N + vacancy (H3) | C (diamond) | yellow |
| Pink Diamond / 粉钻 | N + plastic deformation | C (diamond) | pink |
