# 磷酸盐

## 概述

磷酸盐类是玉石与生物质宝石的主要来源：
- **磷灰石族**：含羟基磷灰石、氟磷灰石、氯磷灰石、磷氯铅矿
- **绿松石**：铜铝磷酸盐，多孔结构（土耳其石的核心致色机理）
- **天蓝石**：与绿松石共生的蓝色磷酸盐矿物
- **磷铝石（巴西石）**
- **独居石、磷钇矿**（含放射性元素）


## 化学式类型

PO₄³⁻ 基团 + 金属阳离子

## 结构

[PO₄]³⁻ 四面体独立或与其他四面体（[OH]⁻、[F]⁻）共顶点连接。金
属阳离子（Ca、Fe、Mn、Al、Li、Be、Cu）与 P 形成稳定配位。
磷灰石族为六方晶系，绿松石为三斜晶系。


## 代表性宝石

| 宝石 | 化学式 | 亚族 |

| Apatite / 磷灰石 | Ca₅(PO₄)₃(F,OH,Cl) | apatite |
| Turquoise / 绿松石 | CuAl₆(PO₄)₄(OH)₈·4H₂O | turquoise |
| Lazulite / 天蓝石 | (Mg,Fe)Al₂(PO₄)₂(OH)₂ | lazulite |
| Brazilianite / 磷铝石 | NaAl₃(PO₄)₂(OH)₄ | brazilianite |
