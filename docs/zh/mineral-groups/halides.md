# 卤化物

## 概述

卤化物在宝石学中以颜色中心（色心）现象著称：
- **萤石（CaF₂）**：可见色心致色的多种颜色（蓝、紫、绿），
  是色心理论的经典研究对象
- **岩盐（NaCl）**：紫色变种称"紫岩盐"，色心致色
- **氟铝石**：罕见的氟铝酸盐宝石


## 化学式类型

MX (X = F, Cl, Br, I)

## 结构

卤素阴离子（F⁻、Cl⁻）与金属阳离子形成离子键。典型结构：萤
石型（CaF₂ 立方）、岩盐型（NaCl 立方）。硬度低、解理完全，
是宝石学的"易碎类"。


## 代表性宝石

|---|---|---|
| Fluorite / 萤石 | CaF₂ | fluorite |
