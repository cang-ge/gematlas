# 硫化物

## 概述

硫化物类宝石以光学效应与金属光泽著称：
- **黄铁矿**：FeS₂，"愚人金"，立方晶系，常用于装饰品
- **闪锌矿**：ZnS，色散极高（0.156），超过钻石
- **方铅矿**：PbS，重且软（Mohs 2.5），少见作宝石
- **辰砂（朱砂）**：HgS，红色，矿物学重要


## 化学式类型

M_xS_y (含 S²⁻)

## 结构

金属阳离子与 S²⁻ 形成离子键或共价键。常见结构包括 NaCl 型
（方铅矿）、闪锌矿型（ZnS 四面体配位）、黄铁矿型（FeS₂ 哑
铃状 S₂²⁻）。


## 代表性宝石

| 宝石 | 化学式 | 亚族 |
|---|---|---|
| Pyrite / 黄铁矿 | FeS₂ | pyrite |
| Sphalerite / 闪锌矿 | ZnS | sphalerite |
