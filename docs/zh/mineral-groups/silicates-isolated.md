# 硅酸盐 · 孤立四面体 / 环状

## 概述

孤立四面体结构（橄榄石型 + 石榴石型 + 锆石型）涵盖多数高折射率
宝石：
- **石榴石族**：铁铝榴石、镁铝榴石、锰铝榴石、钙铝榴石、钙
  铁榴石、翠榴石、变色石榴石
- **橄榄石族**：橄榄石（含镁橄榄石与贵橄榄石）
- **锆石族**：蓝锆石（含天然 U/Th 衰变放射性衰变）
环状硅酸盐中重要宝石：电气石（碧玺）、堇青石（Iolite）。


## 化学式类型

M₃Al₂(SiO₄)₃ / M₃(SiO₄) + Si₃O₉

## 结构

孤立硅酸盐中 [SiO₄]⁴⁻ 四面体不与其他四面体共享氧原子，由
阳离子（M²⁺、M³⁺）在结构空隙中桥接。环状硅酸盐中，3 个
或 6 个 [SiO₄] 四面体首尾连接形成 [Si₃O₉]⁶⁻ 或 [Si₆O₁₈]¹²⁻
环。
