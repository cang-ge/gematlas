# 硅酸盐 · 链状 / 带状

## 概述

链状硅酸盐包含两个重要宝石族：
- **辉石族（单链）**：硬玉（翡翠）、透辉石、蔷薇辉石（部分属
  此）、锂辉石（孔赛石）
- **角闪石族（双链）**：软玉（和田玉）、阳起石、透闪石
翡翠（硬玉）与软玉（透闪石）是中国传统"玉"的两大来源，但
化学与晶体结构完全不同。


## 化学式类型

M·SiO₃ / M₇Si₈O₂₂(OH)₂

## 结构

[SiO₄] 四面体通过共享两个顶角氧连接为单链（辉石）或共享两
个加一个为双链（角闪石）。链间由金属阳离子（如 Ca、Mg、Fe）
和 OH⁻ 桥接。链结构沿 c 轴延伸，使晶体常呈柱状或纤维状习性。


## 代表性宝石

| 宝石 | 化学式 | 亚族 |
|---|---|---|
| Jadeite (Burmese Jade) / 翡翠 | NaAlSi₂O₆ | pyroxene |
| Nephrite / 软玉 | Ca₂(Mg,Fe)₅Si₈O₂₂(OH)₂ | amphibole |
| Spodumene (Kunzite) / 孔赛石 | LiAlSi₂O₆ | pyroxene |
| Spodumene (Hiddenite) / 翠绿锂辉石 | LiAlSi₂O₆ + Cr | pyroxene |
| Diopside / 透辉石 | CaMgSi₂O₆ | pyroxene |
| Rhodonite / 蔷薇辉石 | MnSiO₃ | pyroxenoid |
| Actinolite / 阳起石 | Ca₂(Mg,Fe)₅Si₈O₂₂(OH)₂ | amphibole |
