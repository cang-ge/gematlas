# 单质（原生元素）

## 概述

单质矿物为同种元素原子在自然界结晶形成。宝石界最重要的单质矿
物是金刚石（碳），由 sp³ 杂化的碳原子以四面体共价键形成，是
自然中已知最硬的物质。其他单质宝石（如黄金、白银）多以工艺
品而非刻面宝石出现。


## 化学式类型

单原子或同种原子

## 结构

原子在三维空间按特定几何方式排列形成晶格；金属类（如金、银）
为面心立方结构；非金属（如金刚石）为共价四面体网络。


## 代表性宝石

| 宝石 | 化学式 | 亚族 |
|---|---|---|
| Diamond / 金刚石 | C | non-metal |
| Gold / 自然金 | Au | metal |
| Silver / 自然银 | Ag | metal |
| Platinum / 自然铂 | Pt | metal |
| Copper / 自然铜 | Cu | metal |
