# 硅酸盐 · 架状（网状）

## 概述

架状硅酸盐 SiO₂ 与 Al₂O₃ 摩尔比约为 3:1–6:1。最重要的宝石
矿物族是长石族（月光石 / 拉长石 / 太阳石 / 天河石 / 拉布拉多
拉长石）；次要族为沸石族（不常作宝石）。长石占地球地壳矿物
总重量的约 60%。


## 化学式类型

M(Al,Si)₄O₈

## 结构

[SiO₄]⁴⁻ 四面体每个顶角与其他四面体共享，形成三维无限架状
结构。长石族中 Si⁴⁺ 与 Al³⁺ 在四面体位置有序替代，碱/碱土金
属阳离子充填结构空隙。Si/Al 比与阳离子类型决定矿物种属。
