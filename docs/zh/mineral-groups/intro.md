# 矿物分类组

天然宝石按化学成分与结构可分为多个族。以下是宝石学中最重要的八个组：

## 八个主要组

| 组 | 化学式类型 |
|---|---|
| [单质（原生元素）](native-elements) | 单原子或同种原子 |
| [氧化物](oxides) | M₂O₃ / MO₂ / MO |
| [硅酸盐 · 架状（网状）](silicates-framework) | M(Al,Si)₄O₈ |
| [硅酸盐 · 链状 / 带状](silicates-chain) | M·SiO₃ / M₇Si₈O₂₂(OH)₂ |
| [硅酸盐 · 孤立四面体 / 环状](silicates-isolated) | M₃Al₂(SiO₄)₃ / M₃(SiO₄) + Si₃O₉ |
| [磷酸盐](phosphates) | PO₄³⁻ 基团 + 金属阳离子 |
| [碳酸盐](carbonates) | CO₃²⁻ + 二价金属阳离子 |
| [硫化物](sulfides) | M_xS_y (含 S²⁻) |
| [卤化物](halides) | MX (X = F, Cl, Br, I) |

*参见[分类总览](../classification/intro)了解矿物学分类框架。*
