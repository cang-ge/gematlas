# 氧化物

## 概述

氧化物类是宝石学最重要类之一，涵盖刚玉族（红宝石、蓝宝石）、
尖晶石族、金红石族、水铝矿等。致色元素以过渡金属为主
（Cr、Fe、Ti、V）。因晶格中化学键强（离子键 + 部分共价性），
氧化物硬度普遍较高（Mohs 7–9），是良好的耐久性宝石。


## 化学式类型

M₂O₃ / MO₂ / MO

## 结构

氧离子（O²⁻）构成六方最密堆积或立方最密堆积骨架，金属阳离子
填充八面体或四面体间隙位置。刚玉族为三方晶系，尖晶石族为等轴
晶系。
