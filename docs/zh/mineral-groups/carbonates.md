# 碳酸盐

## 概述

碳酸盐类涵盖宝石界多个富有特色的品种：
- **方解石族**：方解石（含冰洲石）、菱锰矿、菱镁矿、菱铁矿、
  蓝纹石（含 Sr 的蓝色方解石变种）
- **文石族**：文石（霰石）、珍珠母（生物成因）、珊瑚
- **白云石、孔雀石、蓝铜矿**：含水或铜的复杂碳酸盐
碳酸盐普遍硬度较低（Mohs 3–4），多为玉石或有机宝石原料。


## 化学式类型

CO₃²⁻ + 二价金属阳离子

## 结构

[CO₃]²⁻ 平面三角形基团与二价金属阳离子（Ca、Mg、Mn、Fe、Cu、
Zn、Ba、Sr）配位。方解石型为三方晶系；文石型为斜方晶系。
