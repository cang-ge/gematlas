# 三方晶系

## 概述

三方晶系主轴为三重对称（非六重），c 轴垂直方向不存在水平镜面。
晶体为一轴晶。刚玉、石英、电气石是本系最重要的宝石族。晶体双
晶现象普遍，对鉴定有重要价值（如石英的道芬双晶 / 日本双晶）。


## 晶体学参数

| 属性 | 值 |
|---|---|
| 晶轴 | a₁ = a₂ = a₃ ≠ c (rhombohedral subtype) |
| 夹角 | as hexagonal |
| 对称性等级 | medium-high |
| 光学分类 | uniaxial |
| 光性 | + or − (varies) |
| 双折射率范围 | 0.000–0.116 |
| 解理 | variable |
| 常见晶形 | 菱面体、复三方偏三角面体、柱状（带纵纹） |
