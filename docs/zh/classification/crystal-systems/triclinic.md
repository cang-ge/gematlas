# 三斜晶系

## 概述

三斜晶系对称性最低：所有轴两两不垂直，所有角均不为 90°。晶体
为二轴晶。蓝晶石的硬度各向异性（横纹方向 4–5，轴向 6–7）是
三斜结构异性的经典例证。
