# 单斜晶系

## 概述

单斜晶系三条不等长轴，其中两条正交，第三条 b 轴倾斜。晶体为
二轴晶。主要解理平行于 (010) 轴面。软玉（透闪石质）、大多数
长石族矿物属此晶系。


## 晶体学参数

| 属性 | 值 |
|---|---|
| 晶轴 | a ≠ b ≠ c |
| 夹角 | α = γ = 90°, β ≠ 90° |
| 对称性等级 | low-medium |
| 光学分类 | biaxial |
| 光性 | + or − |
| 双折射率范围 | 0.015–0.105 |
| 解理 | pinacoidal (single good cleavage) |
| 常见晶形 | 斜柱状、板状，常出现双晶 |
