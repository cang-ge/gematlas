# 等轴晶系

## 概述

等轴晶系（又称立方晶系）具有七大晶系中最高的对称性。三条等长
晶轴两两正交。等轴晶体在光学上均为均质体——光在各方向传播速度
相同，故无双折射亦无多色性。鉴别主要依靠折射率、比重与光谱测试。
