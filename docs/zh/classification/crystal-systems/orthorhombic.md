# 斜方晶系

## 概述

斜方晶系三条晶轴两两正交但长度互不相等。晶体为二轴晶，具两条
光轴；正负光性因品种而异。本晶系涵盖金绿宝石族、橄榄石族等
多个宝石族系。


## 晶体学参数

| 属性 | 值 |
|---|---|
| 晶轴 | a ≠ b ≠ c |
| 夹角 | α = β = γ = 90° |
| 对称性等级 | medium |
| 光学分类 | biaxial |
| 光性 | + or − |
| 双折射率范围 | 0.040–0.180 |
| 解理 | good parallel to one or two pinacoids |
| 常见晶形 | 斜方柱、斜方双锥、板状 |
