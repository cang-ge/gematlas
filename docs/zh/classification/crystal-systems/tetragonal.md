# 四方晶系

## 概述

四方晶系具有两条等长的水平轴与一条不等长的纵向轴。晶体光学上
为一轴晶，光轴平行于 c 轴；多数宝石为正光性，少数为负光性。
c 轴方向为非寻常光折射率方向。


## 晶体学参数

| 属性 | 值 |
|---|---|
| 晶轴 | a = b ≠ c |
| 夹角 | α = β = γ = 90° |
| 对称性等级 | medium-high |
| 光学分类 | uniaxial |
| 光性 | + |
| 双折射率范围 | 0.040–0.300 |
| 解理 | parallel to base (001) |
| 常见晶形 | 四方柱、四方双锥、板状 |
