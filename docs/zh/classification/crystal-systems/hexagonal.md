# 六方晶系

## 概述

六方晶系（严格按四轴米勒指数）三条水平轴等长、互成 120°，纵向
c 轴与水平轴正交。晶体为一轴晶。绿柱石族以典型六方柱晶形为其
代表。注意：部分参考资料将三方晶系归入六方晶系之下。


## 晶体学参数

| 属性 | 值 |
|---|---|
| 晶轴 | a₁ = a₂ = a₃ ≠ c (4-index notation) |
| 夹角 | 120° / 120° / 120° between lateral axes, 90° vertical |
| 对称性等级 | medium-high |
| 光学分类 | uniaxial |
| 光性 | + or − |
| 双折射率范围 | 0.000–0.180 |
| 解理 | poor / none |
| 常见晶形 | 六方柱、六方双锥 |
