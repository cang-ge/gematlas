# 软玉

<!-- language switcher hint -->
English: [Nephrite (Hetian Jade)](/gems/nephrite)

---

## 分类

| 属性 | 值 |
|---|---|
| 矿物族 | 角闪石族 |
| 化学式 | Ca₂(Mg,Fe)₅Si₈O₂₂(OH)₂ |
| 晶系 | monoclinic |

## 物理性质

| 属性 | 值 |
|---|---|
| 莫氏硬度 | 6.25 |
| 比重 | 2.96 |
| 折射率 | 1.606-1.632 |

## 光学性质

| 属性 | 值 |
|---|---|
| 多色性 | none |
| 典型颜色 | 羊脂白, 青白, 碧绿, 黄玉色, 糖色 |
| 致色原因 | Fe²⁺ 致绿色，Fe³⁺ 致褐黄色 |

## 处理与披露

| 处理方式 | 详情 |
|---------|------|
| 常见方法 | wax-impregnation, dyeing |
| 需披露 | 是 |
| 备注 | 与翡翠（硬玉）属不同矿物种；新疆和田为最著名产地 |
