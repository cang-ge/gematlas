# 💎 GemAtlas — The Open Gemological Compendium

> A bilingual (EN/中文) open-source gemstone knowledge platform, built with VitePress.

[English](#english) · [中文](#中文)

---

## English

**GemAtlas** is a project-level open-source knowledge base covering the full vertical of gemology — from mineralogy and identification to cutting, grading, and luxury maison craftsmanship.

### What's inside (V1.0 MVP)

| Module | Coverage |
|--------|----------|
| **① Classification** | Mineralogical taxonomy, optical phenomena, color trees, crystal systems |
| **② Identification** | Physical properties, Mohs scale, crystal diagrams, inclusions, spectra |
| **③ Cutting** | Faceting geometry, fancy cuts, cabochons, cameos, video walkthroughs |
| **④ Grading** | GIA 4Cs, fancy color grading, treatment disclosure, origin premiums |
| **⑥ Gallery** | 15–20 signature pieces from 7 leading maisons (Cartier, VCA, Boucheron, Tiffany, Harry Winston, Graff, Chaumet) |

10 cornerstone gems in V1.0: Ruby · Sapphire · Emerald · Diamond · Spinel · Tanzanite · Alexandrite · Paraíba Tourmaline · Tsavorite Garnet · Opal.

### Quick start

```bash
pnpm install
pnpm dev      # → http://localhost:5173
pnpm build    # → docs/.vitepress/dist/
```

### Tech stack

- VitePress 1.6 (Vue 3 + Vite)
- TypeScript · MDX
- ECharts · Mermaid
- pnpm · Node 20+

## 中文

**GemAtlas（宝石图典）** 是一个项目级开源宝石知识库，覆盖宝石学完整垂直领域——从矿物学、鉴定到切割、分级与奢侈品工艺。

### V1.0 MVP 模块

| 模块 | 内容 |
|------|------|
| **① 分类** | 矿物分类、光学效应、颜色树、晶系 |
| **② 鉴定** | 物理性质、莫氏硬度尺、晶系图、内含物、吸收光谱 |
| **③ 切割** | 刻面几何、异形切、蛋面、雕刻、视频演示 |
| **④ 分级** | GIA 4Cs、有色宝石分级、处理披露、产地溢价 |
| **⑥ 画廊** | 7 大顶级品牌 15–20 件标志性作品（Cartier、VCA、Boucheron、Tiffany、HW、Graff、Chaumet） |

V1.0 收录 10 种核心宝石：红宝石 · 蓝宝石 · 祖母绿 · 钻石 · 尖晶石 · 坦桑石 · 亚历山大石 · 帕拉伊巴碧玺 · 沙弗莱石榴石 · 欧泊。
